Salon App with Ruby

This Ruby on Rails application is a Salon Booking Application so a user is able to set up an appointment by service and stylist, date and time. This application is modeling MVC (Model, Views, Controller) with RESTful Routing with actions in order to have the view page be produced by the server. This application is using log in/log out through the built in method 'password_digest' through the gem 'bcrypt'. Lastly, I'm using a third party log in/sign up with Google OAuth.
